## Локальний запуск

Для локального запуску необхідний [Node.js](https://nodejs.org/en/).

Ви можете завантажити репозиторій у вигляді zip-архіву, або клонувати цей репозиторій за допомогою командного рядка.

Далі необхідно встановити залежність:

```
npm install
```

Потім додайте команду для можливості редагування слайдів з підтримкою автоматичного перезавантаження змін:

```
npm run start
```
