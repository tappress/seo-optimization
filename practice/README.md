# SEO оптимізація та просування (практичні завдання)

## Список практичних завдань

### Лабораторна робота №1. Вступ до SEO та пошукових систем

- Розгортання тестового сайту (існуючий проєкт з попереднього курсу "Фрейворки JavaScript")
- Вибір домену та хостингу
- Підключення Google Search Console
- [Методичні вказівки](lab-01) до виконання лабораторної роботи

### Лабораторна робота №2. Індексація та алгоритми Google

- Перевірка індексації сторінок
- Аналіз Coverage Report
- URL Inspection

### Лабораторна робота №3. Семантичне ядро та структура сайту

- Збір ключових слів
- Кластеризація запитів
- Побудова структури сайту

### Лабораторна робота №4. Контент і on-page SEO

- Оптимізація сторінки
- Написання SEO-тексту
- Перевірка релевантності

### Лабораторна робота №5. Внутрішня перелінковка

- Аудит перелінковки
- Побудова схеми перелінковки

### Лабораторна робота №6. Технічне SEO

- Технічний аудит сайту
- Виправлення технічних помилок

### Лабораторна робота №7. Core Web Vitals та швидкість

- Аналіз швидкості
- Оптимізація Core Web Vitals

### Лабораторна робота №8. Зовнішня оптимізація та лінкбілдінг

- Аналіз backlink профілю
- Побудова link strategy

### Лабораторна робота №9. Поведінкові фактори та UX

- UX-аудит сайту
- Аналіз поведінкових показників

### Лабораторна робота №10. Веб-аналітика та SEO-стратегія

- Налаштування GA4
- Фінальний SEO-аудит проєкту
